# 🚀 AI Skill Gap Analyzer

## 📌 Problem

Many students are unsure about the skills required for their desired job roles. There is no simple tool that clearly identifies missing skills and provides a structured learning path.

---

## 💡 Solution

AI Skill Gap Analyzer is a web application that:

* Analyzes user-entered skills (resume input)
* Compares them with job role requirements
* Identifies missing skills
* Generates a personalized learning roadmap
* Calculates a match score percentage

---

## ⚙️ Features

* ✅ Skill extraction from user input
* ✅ Missing skill detection
* ✅ Personalized roadmap generation
* ✅ Match score calculation
* ✅ Handles unknown job roles
* ✅ Interactive and animated UI

---

## 🧠 How It Works

1. User enters skills and job role
2. System extracts known skills
3. Compares with required job skills
4. Identifies missing skills
5. Generates a roadmap

---
